# 📊 Análisis de Evasión de Clientes (Churn) - Telecom X

## 🎯 Propósito del Análisis
El propósito de este proyecto es diagnosticar y mitigar la alta tasa de cancelación (*churn*) que enfrenta la empresa de telecomunicaciones Telecom X. Como Analista de Datos, el objetivo abarca dos frentes:

1. **Negocio:** Descubrir los patrones y motivos principales de fuga de clientes mediante un Análisis Exploratorio de Datos (EDA) para proponer estrategias de retención accionables.
2. **Ciencia de Datos:** Ejecutar un pipeline de Extracción, Transformación y Carga (ETL) para entregar un dataset íntegro, limpio y estructurado numéricamente, listo para que el equipo de Machine Learning entrene modelos predictivos.
