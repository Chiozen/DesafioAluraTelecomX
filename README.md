# DesafioAluraTelecomX
Solución al desafío Telecom X de Alura LATAM. Proyecto de Data Science para analizar la evasión de clientes (churn). Incluye pipeline ETL (extracción de API y limpieza) y Análisis Exploratorio (EDA) con Python y Pandas para descubrir patrones de fuga y preparar los datos para Machine Learning.
